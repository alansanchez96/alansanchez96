## 🧩 Featured Project: InvitaSR

**InvitaSR** is a multitenant SaaS platform for creating, publishing, and managing digital invitations for events.

The platform includes:

- Laravel 12 backend
- Vue.js 3 + TypeScript frontend
- Multitenancy by database
- Master database for clients, auth, plans, payments, subscriptions, templates, countries, onboarding, credits, logs, and subdomains
- Isolated tenant databases for invitations, RSVP responses, wall messages, gallery images, text overrides, and event data
- Stripe and MercadoPago payment integrations
- Public and private onboarding flows
- Template-based invitation creation
- Event subdomains
- Admin backoffice
- Client dashboard
- PDF/XLSX exports
- WhatsApp Cloud API integration
- AI-assisted development workflows with LLMs, MCP, Copilot and Codex

This project reflects my ability to design and build a real SaaS product from architecture to implementation.
